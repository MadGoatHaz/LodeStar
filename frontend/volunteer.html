How to Become a Volunteer Crawler

1

Generate Your Crawler Key Pair

Run these commands on your volunteer machine:

openssl genpkey -algorithm RSA -out crawler.key -pkeyopt rsa_keygen_bits:2048
openssl rsa -pubout -in crawler.key -out crawler.pub
Important: Never share your crawler.key file. This is your private key and must be kept secret.
2

Share Your Public Key

Send the contents of your crawler.pub file to the Lodestar team for verification:

cat crawler.pub

Copy the entire output (including -----BEGIN PUBLIC KEY----- and -----END PUBLIC KEY-----) and submit it through our secure channel.

3

Install and Run the Crawler

Install dependencies and run the crawler:

pip install -r requirements.txt
python crawler/crawler.py

Your crawler will now collect and sign data, storing it directly on IPFS for verification by the main site.

4

Verification Process

The main Lodestar site verifies all data using your public key. Only signed data from verified crawlers will appear on the site.

Security Note: Data is signed before storage on IPFS, ensuring no modifications can occur after signing.

Security and Trust

Our volunteer program maintains the highest security standards:

  • Private Keys Stay Private: Your crawler.key never leaves your machine
  • Public Keys Verified: Only keys on the official trusted_keys.json are accepted
  • Signature Verification: All data is verified using cryptographic signatures
  • No Central Authority: Data is stored on IPFS with no single point of failure

For more technical details on how verification works, see our implementation guide.
